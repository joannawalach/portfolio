ABOUT ME

Html, css, java script, react

Programowaniem interesuję się od wielu lat. Początkowo traktowałam je jako hobby. Ostatni gwałtowny rozwój technik front-endu zachęcił mnie do pogłębienia i usystematyzowania mojej wiedzy programistycznej.

Swoją przygodę z Front-endem rozpoczęłam od tworzenia statycznych stron www oraz stron opartych na popularnych systemach zarządzania treścią, tj.: Wordpress, Joomla. Biegle korzystam z programów graficznych - Corel, Photoshop, w zakresie tworzenia grafiki na potrzeby stron www oraz dtp.

W chwili obecnej moje zainteresowania koncetrują się wokół Java Script oraz Reacta, w perspektywie ich wykorzystanie do tworzenia aplikacji komunikujących się z bazami danych.
